# 개발일지 2026-07-01

## 랜딩 파비콘 흰 배경 제거 (투명 배경 재생성)

### 배경
- 랜딩 파비콘이 "Nuxt 기본"으로 보인다는 제보 → 서버 검증 결과 실제로는 브랜드
  파비콘 정상 배포, **브라우저 파비콘 캐시** 문제로 확인 (라이브 md5=로컬 일치).
- 후속 확인: 파비콘에 **흰색 사각 배경이 불투명하게 구워져** 있었음 (모서리 alpha=255).
  기존 realfavicongenerator 입력 이미지가 흰 배경 포함본이었던 게 원인.

### 조치
- 원본 `assets/images/section1_symbol.png`(투명본, 478×568)에서 파비콘 전체 재생성.
- 도구: ImageMagick/PIL/sharp 부재 → macOS `sips`로 568×568 투명 정사각 패딩 후 크기별 리사이즈.
- `favicon.ico`는 16/32/48 멀티사이즈 PNG-in-ICO로 교체.
- public/ 아이콘 25개 교체(favicon/android/apple/ms), 전 파일 모서리 alpha=0 검증.
- 캐시 재발 방지로 `nuxt.config` 아이콘 href에 `?v=2` 부여.

### 트러블슈팅
- 1차 배포 실패: `?v=2`를 `rel="manifest"` href에도 붙여 Nuxt 프리렌더 크롤러가
  `/manifest.json?v=2`를 라우트로 오인 → 404 → 빌드 실패.
- 교훈: **Nuxt manifest href에는 쿼리스트링 금지.** manifest만 원복 후 재배포 성공.

### 결과
- www.lighthouse.career 파비콘 투명 배경 반영 완료·검증 (CloudFront 무효화 Miss 확인).
- 잔여(경미): iOS apple-touch-icon은 투명 시 홈화면에서 검정 배경 합성됨. 필요 시 apple만 배경색본 별도 생성.
