## [인프라] AWS + 홈서버 하이브리드 아키텍처 설계

### 배경
- 동시접속 20명 규모의 안정적인 배포 환경 검토
- AWS 단독 vs 홈서버 활용 비용 비교

### 결정된 아키텍처

```
lighthouse.career         → Nginx → PM2 (Node.js BE :3000)  ← 홈서버
admin.lighthouse.career   → Nginx → Next.js Admin :3001      ← 홈서버 (예정)
(FE)                      → AWS S3 + CloudFront              ← AWS (예정)
DB                        → MongoDB Atlas M0                  ← 무료
```

**예상 월 비용: ~$3–5** (AWS S3/CloudFront + Route 53만)

### 비교 검토한 옵션
| 옵션 | 구성 | 월 비용 |
|------|------|---------|
| A | 단일 EC2 + S3/CloudFront | ~$20 |
| B | EC2 분리 + ALB + S3/CloudFront | ~$55 |
| C | ECS Fargate | ~$60 |
| **채택** | **홈서버(BE+Admin) + S3/CloudFront(FE)** | **~$3–5** |
