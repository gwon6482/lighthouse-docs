## [랜딩] 비즈니스 제안 페이지 (/business)

준비중 페이지였던 `/business`를 실제 B2B 제안 페이지로 구현.

- **디자인**: 다크 테마(거의 블랙 #0b0d12) + 진지한 분위기. 히어로에 라이트하우스 등대 심볼(노란 원) + 글로우/빛줄기 효과. 노란색(브랜드) 포인트.
- **폼**: 회사/기관명, 담당자명, 연락처, 이메일, 제안유형(select), 제안내용, 개인정보 동의 + 클라이언트 검증 + 성공/에러 UI + honeypot 스팸방지.
- **제출 연동**: Web3Forms(무료, 백엔드 불필요·정적 호환). `POST api.web3forms.com/submit`, reply-to=제안자 이메일. 제출 내용이 등록 수신메일로 발송됨. E2E 테스트(제출→200→메일 수신) 통과.
- ※ access_key는 공개값(클라이언트 노출 무방, 등록 메일로만 발송).

**배경/구조**: test(워크벤치)와 app(본서비스)을 한 코드베이스로 운영. 기능 코드 1벌을 두 셸이 공유.
