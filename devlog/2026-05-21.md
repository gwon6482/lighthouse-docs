## [BE] 목표 진로 API 구현 + 배포

**파일**: `models/User.js`, `controllers/userController.js`, `routes/user.js`

### User 스키마 추가
```js
targetCareer: {
  refType: { type: String, enum: ['jobCode', 'custom'] },
  ref:     { type: String }
}
```
- `jobCode`: 진로백과(job_info) 직업 참조
- `custom`: 진로백과에 없는 사용자 지정 진로

### 신규 엔드포인트
| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET | `/api/user/target-career` | 조회 (jobCode면 job_info join) |
| PUT | `/api/user/target-career` | 설정/변경/삭제 (null → 삭제) |

**배포**: main 브랜치 push → GitHub Actions 자동 배포 완료
