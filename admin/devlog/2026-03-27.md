- NextAuth v5 JWT 기반 인증 구현 (Credentials Provider)

## 버그 수정

### 사이드바 active 탭 항상 대시보드로 고정되는 문제
- **원인**: `startsWith("/dashboard")` 로직으로 인해 하위 경로에서도 대시보드 탭이 active 처리됨
- **수정**: 대시보드 탭은 `pathname === "/dashboard"` 정확히 일치할 때만 active

### Hydration mismatch 에러
- **원인**: Server Component에서 Sidebar를 SSR로 렌더링할 때 pathname 컨텍스트 불일치
- **시도 1**: `mounted` state로 클라이언트 마운트 후에만 active 계산 → 해결 안 됨
- **최종 해결**: `SidebarWrapper` (Client Component)를 통해 `ssr: false`로 Sidebar를 클라이언트 전용 렌더링
  - `ssr: false`는 Server Component에서 직접 사용 불가 → Client Component 래퍼 필요
